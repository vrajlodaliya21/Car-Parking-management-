
#🚗 Car Parking Management System

A web-based Car Parking Management System developed using PHP and MySQL that helps manage vehicle parking operations efficiently with separate Admin and User roles.

📌 Features
👤 User Module

User registration and login

Vehicle entry and exit management

View parking details and status

🛠️ Admin Module

Secure admin authentication

Dashboard with real-time statistics

Manage users and vehicle records

Parking slot monitoring

Full CRUD operations (Create, Read, Update, Delete)

🧰 Technologies Used

Frontend: HTML5, CSS3, Bootstrap

Backend: PHP

Database: MySQL

Server: Apache (XAMPP / WAMP)

🗂️ Project Structure
car-parking-management/
│
├── admin/              # Admin panel files
├── user/               # User panel files
├── includes/           # Database & common files
├── assets/             # CSS, JS, images
├── database/           # SQL file
├── index.php           # Main entry point
└── README.md

⚙️ Installation & Setup

Clone the repository

git clone 


Move project to server directory

For XAMPP:

C:\xampp\htdocs\


Create Database

Open phpMyAdmin

Create a database (e.g. parking_db)

Import the provided .sql file from the database/ folder

Configure Database Connection

Update database credentials in:

includes/config.php


Run the project

Open browser and visit:

http://localhost/car/Index.php
